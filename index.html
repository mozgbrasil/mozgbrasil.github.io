Benefits

Take control of your tags.

No need to ask the IT team to change tags again; our user-friendly tool lets you manage your own website and mobile app tags on the spot. So you can focus on marketing, not coding.

  • Robust

    Manage tags reliably.

    Tag Manager runs on Google’s infrastructure for security, speed, and performance. Tools to manage your own processes are included, so you can make sure things are working just fine within your own infrastructure.

  • Intuitive

    Change tags with ease.

    With Google Tag Manager containers in place, you can track behavior on your site in almost endless ways. Add or change tags whenever you like; your IT team will be reassured that your site is loading and running smoothly.

  • Open

    Plays nicely with others.

    Marketers can add or change their own tags as needed. Google Tag Manager supports a wide range of Google tags and a variety of third-party provider tags – for web and mobile apps. Your campaign can be underway with just a few clicks. Don’t see a tag listed? You can add it immediately as a custom HTML tag.

Built for Mobile Apps

Google Tag Manager works with Firebase, Google's platform for mobile developers, to make it easier than ever for marketing and development teams to work together. Developers can use Firebase Analytics events to measure everything that’s happening in an app. Then, using Tag Manager’s easy to use web interface, marketers can modify and route these events to various measurement solutions — including Firebase Analytics, Google Analytics, Kochava, Tune, Adjust, and more. Updates in Google Tag Manager are sent to your installed apps, allowing teams to update their measurement setup on the fly.

Learn more

Customizable API

Tailor to your business with the Google Tag Manager API.

Build custom solutions to fit your needs with Google Tag Manager’s API. It offers all the tools of the web interface, like publishing containers and tags, and makes it easy to manage user access in bulk. It’s also a great way for agencies to manage large tagging setups for clients.
